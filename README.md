# object-oriented-programming

## GitHub Pages

This repository is set up with a small static `index.html` page and a GitHub Actions workflow that deploys it to GitHub Pages.

To publish it:

1. Push these files to the `main` branch on GitHub.
2. Open the repository on GitHub and go to `Settings` > `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.
4. Go to the `Actions` tab and run `Deploy GitHub Pages`, or push another commit to `main`.

After the workflow finishes, GitHub will show the published site URL in the workflow summary and in `Settings` > `Pages`.
